# MiniGameWorld-Controller
- The OPs can control almost things of MiniGameWord with players (Non-OP players are under control by the OPs)

# Rules
- Non-OP players can NOT use any commands of MiniGameWorld (but can use sign block (Sign block will be also blocked with permission))
- MiniGame will be only started with the OPs command (`/mwc start [<minigame>]`)

# Commands
- `/mwc start [<minigame>]`: start <minigame> minigame
- `/mwc finish [<minigame>]`: finish <minigame> minigame
- `/mwc <join | leave | view | unview> <minigame> [<player> <player> <player> ...]`: Control non-op or specific players
